NoobNote

A Beginner Friendly Python Project

Intro:

NoobNote is my attempt to make an almost perfect clone of Microsoft Notepad in python. It has all the functions of Notepad. If compiled, it can perfectly replace Notepad. I wrote NoobNote to practise tkinter, python and also to brush up my html skills while making this website. This is just my fun take on Notepad. NoobNote is not compiled by deflaut. Although it does come with a easy script you can run to compile it into an .exe file. NoobNote is not compiled because it is not yet a fully develloped app. It is just my fun take on NotePad. Hence you need to install python to use NoobNote

Post Installation:

Once you unzip the file, you get the following contents
1. NoobNote.py is the base python file. You can either launch it from the terminal with "python NoobNote.py" or just double click the file to launch NoobNote
2. NoobNote.pyw to open the python file without terminal and is handy for regular use.
3. NoobNote_explained.py is explaination of NoobNote Code. Available from v.0.2
4. README.txt for the docs. You can also choose the docs option in NoobNote for reading
5. trobuleshoot.py checks if all modules are installed and helps you trobuleshoot
6. compile.bat compiles NoobNote in Windows
7. compile.sh compiles NoobNote in Linux
8. icon.ico for the icon
9. NoobNoteStart.bat is script to run NoobNote on Windows.

As you can see, the file structure is pretty simple and it shouldn't be difficult to use it once you get used to it. You can also customize this to your liking
