Which one Should I Download

NoobNote comes in many different flavours. NoobNote is a base on top of which you can build any feature you want to as long you know a little python. I have made four NoobNote flavours which each have their own use cases and purposes. NoobNOte is registered under the MIT License, hence you are free to use it as you like and make your own version so of NoobNote, although I am not responsible for your product. See the four options you have below and for more information about these flavours, go here.

LiteNote

LiteNote in my my opinion is the best one for writers and people who want more functionality while at the same time maintianiing a neat and nostalgic feel. LiteNote comes with many features such as an in built toolbar, printing support, favourite files and nifty little features like a Calculator, Qrcode generator, finder, Clock and Url shortener among others. So if you are like feature rich and customizable applications, this one's for you.

To Download LiteNote v.3.1 See below

NoobNote

NoobNote is a simple vanilla version of the Noobote base. You can think of NoobNote as an example of what you can do with NoobNote base. You can literally do anything with it. But if you enjoy a plain and simple version with no external tools and rather optional features, this one's for you.

To Download NoobNote v.3.1 See below

DevNote

DevNote is one of the best ones as an alternative to Notepad. It has all the features of LiteNote except the toolbar. It comes with print functionality, code runner, a command palate, finder, settings gui and a url shortener. DevNote also has a nostalgic feel and looks and feels like Notepad. It is very fast and can be opened in multiple windows. It also has the option to favourite a file and open it directly. It is in short, the one for you if you like to take quick notes and like things fast while at the same time maintianing functionality. DevNote also has a code runner for many popular languages.

To Download DevNote v.3.1 See below

CompactNote

CompactNote is basically a onefile distribution of NoobNote, light weight, fast and portable. Only 72mb and easy to just put it on you usb and forget about it. Exact copy of Notepad in terms of functionality, nothing extra, nothing heavy. If you like portable applications, this one's for you.

To Download CompactNote v.0.1 See below

PyPi Install

NoobNote Vanilla can be easily installed with pip. If you have python installed on your system, you can easily install NoobNote using pip, the python package manager. After that, you can open NoobNote from the terminal easily. There are many advantages of using NoobNote from pypi. It is super fast, light weight and you can quickly open it from the terminal and write a small quick note.

$ NoobNote

pip install NoobNote
